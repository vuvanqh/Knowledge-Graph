# Knowledge Graph

A distributed system for transforming unstructured data into a structured, queryable knowledge graph enhanced with semantic search and LLM reasoning.

## Features
- Document ingestion pipeline
- Vector search (semantic retrieval)
- Knowledge graph representation
- RAG-based querying
- ML/LLM integration

- React (frontend)
